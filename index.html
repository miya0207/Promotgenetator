📖 使い方ガイド

    
基本の流れ

    
    
      
  1. 左フォームに情報を入力する
    2. 
      
  2. 「プロンプトを生成する」を押す
    3. 
      
  3. 右パネルのプロンプトをコピー or ダウンロード
    4. 
      
  4. Claude にペーストしてノート本文を生成
    5. 
    

    
ノートタイプの使い分け

    
    
      
  • free / intro / concept — コマンド・設定値なしの概念解説ノート
    • 
      
  • paid / howto / ref — 実装手順・地雷マップ・復旧ルートを含む実践ノート
    • 
    

    
タグ入力の操作

    
    
      
  • テキストを入力して Enter → タグ追加
    • 
      
  • Backspace（空欄時）→ 最後のタグを削除
    • 
      
  • タグの × → そのタグを削除
    • 
    

    
キーボードショートカット

    
    
      
  • Ctrl + Enter — プロンプトを生成
    • 
      
  • Ctrl + Shift + C — コピー
    • 
      
  • Escape — このガイドを閉じる
    • 
    

    
シリーズモード

    
ヘッダーの「シリーズ」タブに切り替えると複数ノートをまとめて設定できます。各ノートカードをクリックすると詳細（goals・must_include等）をインラインで編集できます。生成後は全ノートを一括ダウンロード（ZIP）できます。

    
入力の永続化

    
フォームの入力内容はブラウザセッション中は保持されます。ページを再読み込みしても復元されます（リセットボタンで消去）。
